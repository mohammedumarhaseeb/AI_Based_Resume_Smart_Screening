Understanding the Physics

Gravity: The Pull of Earth

Imagine Earth as a giant magnet that pulls everything toward its center. Sir Isaac Newton described this with his Law of Universal Gravitation. The heavier an object is, and the closer it is to Earth, the stronger the pull.

F = G * (m1 * m2) / r²

Where G is the gravitational constant, m1 and m2 are masses, and r is the distance.

Achieving Levitation ("Antigravity")

True "antigravity" (turning off gravity completely) belongs to science fiction. However, we can achieve levitation by matching gravity's downward pull with an equally strong upward push. This upward force is called the Opposing Force.

Magnetic Levitation

Like poles of a magnet repel each other. Trains like the Maglev use powerful electromagnets to push off the track and float, bypassing friction.

Electrostatic Force

Using high voltages, static electricity can be used to repel extremely light weight materials by charging them identically to a base pad.

The Force Balancing Decision

The physics rule to decide if an object lifts is simple:

IF (Opposing Force > Gravitational Force) THEN { Object Lifts = TRUE }
